# Paper Card: QASPER
Source title: A Dataset of Information-Seeking Questions and Answers Anchored in Research Papers
Source URL: https://arxiv.org/abs/2105.03011
License: CC BY 4.0 via the arXiv license link.
Attribution: Dasigi, Lo, Beltagy, Cohan, Smith, and Gardner, 2021.
Derived note: This card paraphrases source metadata and abstract details for file-reading examples; it is not the full paper.

Study focus: QASPER is a dataset for question answering over full research papers.
Dataset scale: QASPER contains 5,049 questions over 1,585 NLP papers.
Question creation: Each question was written after the question writer read only the title and abstract.
Evidence requirement: Separate answer writers supplied answers plus supporting evidence.
Evaluation lesson: The paper motivates document-grounded, information-seeking QA because existing models lagged human performance on whole-paper questions.
Good Contract2Agent task fit: Use QASPER-style examples for citation_required_qa and unanswerable_question tasks.
